## 两个事业部对比

| 指标 | 医疗事业部 | 再生事业部 |
|---|---|---|
| MQL → SQL | 30% | 80% |
| SQL → 成交 | 20% | **3%** |
| 评估 | 正常 | **异常偏低** |

## 再生事业部问题诊断

### 核心问题

SQL → 成交转化率仅 3%（正常应为 ~20%），说明线索交接和销售跟进存在严重问题。

### 根因分析

1. **语言障碍** — 销售团队不会西班牙语，无法通过视频通话方式跟进南美市场线索
2. **线索交接流程缺失** — 市场部到销售部的交接文档不规范
3. **CRM 记录不完整** — 线索追踪和活动记录不充分
4. **交接方式差异** — 不同市场/区域的交接方法不统一

### MQL/SQL 定义问题

MQL → SQL 转化率 80% 异常高，可能是 MQL/SQL 定义标准过于宽松，需要重新定义。

## 行动计划

1. 从 CRM 中抽取 3-5 个典型客户案例，逐条分析交互记录和邮件

3. 让销售团队负责人 Coral 参与分析讨论
